# Security Policy

## Reporting a vulnerability

If you discover a security issue in this project, please report it responsibly.

**Do not open a public issue.**

Instead, contact the maintainer directly via the email associated with the GitHub account listed in [CODEOWNERS](CODEOWNERS).

## Scope

This is a demonstration project, not production software. However, we take security seriously and will address reported issues promptly.

## Supported versions

Only the latest commit on `main` is supported.
